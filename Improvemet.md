# 🔶 3. Typography Refinement

Your typography is good — but not premium yet.

### Upgrade:

### Headline

Use:

* Tighter line height
* Bigger contrast between bold and regular
* Slight negative letter spacing on big headlines

Example:

* Main headline: 64–72px
* Subheadline: 18–20px
* Tracking reduced slightly

---

### Secondary Labels

Right now:
uppercase tracking-widest everywhere

This is strong but slightly overused.

Refine by:

* Using lighter gray
* Reducing opacity
* Decreasing tracking slightly

Make it less aggressive.
